### Что это за приложение?

#### Короткий ответ
Интерактивное приложение для изучения тем через вопросы и ответы.

#### Развернутый ответ
**Приложение помогает изучать различные темы с помощью карточек.**

##### **Как это работает:**
1. Каждая тема — отдельный `.md` файл
2. Вопросы начинаются с `### `
3. Ответы делятся на короткие и развернутые
4. Можно скрывать ответы для самопроверки

##### **Чтобы начать:**
1. Добавьте свои `.md` файлы в папку `cards/`
2. Используйте правильный формат вопросов
3. Выберите тему слева и начинайте обучение


##### **Формат файла:**
```
### Ваш вопрос?
#### Короткий ответ
Кратко...
#### Развернутый ответ
Подробно...
```
